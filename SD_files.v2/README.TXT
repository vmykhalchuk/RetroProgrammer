This folder contains SD card example files and templates.
All files are ASCII Western Latin 1, and names must be 8.3 DOS style.
NB: SD should be formatted with FAT-16 (FAT-32 not supported!)

CONFIG.TXT - is general configuration
*.HRP      - are HEX files containing EEPROM/Flash/Conf/...

HRP - stands for HEX Retro Programmer - contains Target MCU EEPROM/FLASH/Conf data
